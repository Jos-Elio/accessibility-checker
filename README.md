# Color Accessibility Checker

A WCAG contrast checker and Color Vision Deficiency (Color Blindness) simulation tool.
Built to make accessible design easier for designers and developers.

## Deployment

🔗 [Live Demo](https://accessibility-checker-chi.vercel.app/)

## Features

- **Contrast Ratio Check** – real-time WCAG 2.1 AA and AAA compliance check for normal and large text
- **Live Preview** – see your color combination as text on background, and the reverse
- **CVD Simulation** – visualize how your colors appear for people with Deuteranopia, Protanopia, and Tritanopia
- **CSS Export** – copy your color combination as CSS custom properties
- **EN / DE** – English and German language support
- **Responsive** – works on mobile and desktop

## Tech Stack

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Claude](https://claude.ai/) (Anthropic) – AI coding companion for debugging and guidance

## Motivation

Accessibility has always been part of good design. As a graphic designer transitioning into frontend development, I built this tool to combine both disciplines — and to contribute something useful to designers and developers who care about inclusive digital experiences.
